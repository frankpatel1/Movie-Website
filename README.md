# Movie Website

A modern and responsive movie discovery website designed to showcase films, trailers, and entertainment content through an engaging user interface. This project demonstrates front-end development skills, responsive layouts, and interactive web design techniques for media and entertainment platforms.

## Features

* Fully responsive design
* Movie showcase and featured content sections
* Modern and intuitive UI/UX
* Smooth navigation and interactive elements
* Mobile-friendly experience
* Clean and organized codebase

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/frankpatel1/Movie-Website.git
   ```
2. Open the project directory.
3. Launch `index.html` in your browser.
